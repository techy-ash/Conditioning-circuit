# Conditioning-circuit
For Synthetix 4.0

The objective of this project is to design and simulate an analog signal conditioning circuit capable of extracting a useful signal component from a composite analog input signal generated by a black-box source.

The input signal contains multiple deterministic interference components and broadband noise that simulate an electrically hostile industrial environment. The circuit must therefore suppress unwanted frequency components and noise while ensuring that the desired signal dominates the output.
